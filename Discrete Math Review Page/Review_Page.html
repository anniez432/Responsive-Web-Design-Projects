Asymptotics

Asymptotic growth:

how fast the output f(n) grows as input n grows

Big-O: upper-bound;

f(n) is O(g(n)) if there's constant c, n0 such that f(n)<=c*g(n) for n > n0; f(n) grows at most as fast as g(n)

Big-Omega: lower-bound;

f(n) is Omega(g(n)) if there's constant c, n0 such that f(n)>=c*g(n) for n > n0; f(n) grows at least as fast as g(n)

Big-Theta:

f(n) is theta(g(n)) if it is O(g(n)) and Omega(g(n))

Algorithm Complexities

# of atomic operations performed with a loop:

f(n)=c*n + d = theta(n)

c: constant # of operations per iteration

d: constant # of operatuons performed before & after the loop

Induction

  1. State you're proving by induction. Define P(n) in terms of the variable

  2. Base case - prove P(initial #) holds

  3. Inductive step - Inductive hypothesis assumes P(k) holds, use it to show that P(k+1) also holds

  4. Conclude we've proved statement by induction for all n>=0

Correctness Iterative

  1. Loop Invariants: prove the given ones (i.e. relationship between variables that doens't change, use induction on the # of loop invariants

  2. Partial Correctness: Assume input is valid and program terminates, use loop invariants to argue program returns correct result

  3. Termination: use loop invariants to argue loop terminates, consider quantity that decreases each iteration, so eventually loop will stop when exit condition is met.

Correctness Recursive

1) Partial Correctness: Assume input is valid and program terminates, show correct result is returned w/out recursive calls, show input to recursive call is valid, assume recursive call returns correct result & show then that whole program returns the correct result.

2) Termination: show the chain of calls ends, use induction on a quantity that decreases with each call.
